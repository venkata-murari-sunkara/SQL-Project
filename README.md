**New Wheels – SQL Business Analysis Project**
Quarterly Business Performance Report Using SQL
This project analyzes the **business performance of New-Wheels**, a pre‑owned vehicle resale company that has been **experiencing declining sales and customer satisfaction**. Using SQL, the **project answers key business questions**, uncovers trends, and provides actionable insights for leadership decision‑making.

**Project Overview**
New-Wheels launched an app that manages the entire resale process—from listing vehicles to shipping and collecting customer feedback. Over the past year, sales and customer ratings have steadily declined.
The CEO requested a quarterly business report to understand:
- Customer distribution
- Vehicle preferences
- Revenue trends
- Shipping delays
- Customer satisfaction patterns
This project uses **SQL queries** to analyze the **company’s database and generate insights**.

**Dataset Description**
The analysis uses **multiple tables** from the company’s database:
- customer_t – Customer demographics and credit card details
- order_t – Orders, dates, shipping info, feedback, pricing, discounts
- product_t – Vehicle details (maker, model, price, etc.)
